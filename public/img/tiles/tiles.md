# Naming
naming according to edges in the following scheme:
```
<Edge><Edge><Edge><Edge>(<Number>)?(_<Special>)?
```
Edges are named starting from the top side and then going clockwise.

Edges:
- C: City
- R: Road
- F: Field

Special:
- M: Monastery
- S: Coat of Arms
